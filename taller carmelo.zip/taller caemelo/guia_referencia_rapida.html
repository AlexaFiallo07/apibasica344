Ejercicios de Apropiación - Conceptos Clave y Métodos

🔍 Selectores y Obtención de Elementos

getElementById()

Obtiene un elemento por su atributo id

const elemento = document.getElementById('miId');

✓ Rápido y específico

✗ Solo un elemento a la vez

querySelector()

Obtiene el PRIMER elemento que coincida con un selector CSS

const elemento = document.querySelector('.clase'); const elemento = document.querySelector('#id');

✓ Selectores CSS complejos

✗ Solo el primer resultado

querySelectorAll()

Obtiene TODOS los elementos que coincidan con un selector CSS

const elementos = document.querySelectorAll('li'); elementos.forEach(el => console.log(el));

✓ Múltiples elementos

✓ Selectores complejos

getElementsByClassName()

Obtiene elementos por su clase (HTMLCollection)

const elementos = document.getElementsByClassName('miClase');

✓ Devuelve colección "viva"

✗ Menos flexible que querySelector

🛠️ Creación y Modificación del DOM

createElement()

Crea un nuevo elemento HTML

const nuevoDiv = document.createElement('div'); const nuevaLista = document.createElement('li');
💡 Consejo: No aparece en el DOM hasta que lo añadas con appendChild o insertBefore

appendChild()

Agrega un elemento hijo al final del elemento padre

lista.appendChild(nuevoItem); contenedor.appendChild(tabla);
💡 Consejo: El elemento se mueve si ya existía en otro lugar del DOM

createTextNode()

Crea un nodo de texto puro (sin HTML)

const texto = document.createTextNode('Hola mundo'); elemento.appendChild(texto);
💡 Consejo: Más seguro que innerHTML para textos sin HTML

remove()

Elimina un elemento del DOM

elemento.remove(); // O desde el padre: padre.removeChild(elemento);
💡 Consejo: remove() es más simple que removeChild()

🎨 Manipulación de Atributos y Estilos

setAttribute()

Asigna un atributo a un elemento

elemento.setAttribute('class', 'miClase'); elemento.setAttribute('id', 'miId'); elemento.setAttribute('data-info', 'valor');
✓ Moderno: Usado comúnmente en código actual

classList

Gestiona clases CSS de forma segura

elemento.classList.add('activo'); elemento.classList.remove('inactivo'); elemento.classList.toggle('destacado'); elemento.classList.contains('visible');
✓ Recomendado: Mejor que manipular className directamente

style (Estilos en línea)

Aplica estilos CSS directamente

elemento.style.color = 'red'; elemento.style.display = 'none'; elemento.style.backgroundColor = '#ff0000';
💡 Nota: Evita cuando sea posible, usa clases en su lugar

innerHTML vs innerText

// Interpreta HTML elemento.innerHTML = '<b>Negrita</b>'; // Solo texto elemento.innerText = '<b>Negrita</b>';
⚠️ Cuidado: innerHTML con datos no seguros puede provocar XSS

⚡ Eventos Importantes

Evento Cuándo se dispara Uso común
click Cuando se hace clic en un elemento Botones, enlaces interactivos
input Mientras se escribe en un input Búsqueda en tiempo real
change Después de cambiar un valor y perder el foco Selects, checkboxes
submit Cuando se envía un formulario Validación de formularios
mouseover Cuando el mouse pasa sobre un elemento Efectos hover
focus Cuando un elemento obtiene el foco Campos de formulario

📋 Patrones Comunes

Patrón 1: Agregar un elemento a una lista

1
Crear: const nuevoItem = document.createElement('li');
2
Contenido: const texto = document.createTextNode('Mi item');
3
Agregar contenido: nuevoItem.appendChild(texto);
4
Insertar en DOM: lista.appendChild(nuevoItem);

Patrón 2: Iterar y crear múltiples elementos

1
Array de datos: const datos = [{...}, {...}];
2
forEach: datos.forEach(item => { ... });
3
Dentro del forEach: Crear elemento, agregar contenido, insertar

Patrón 3: Filtrar elementos visibles

1
Input evento: buscador.addEventListener('input', () => {});
2
Obtener valor: const busqueda = buscador.value.toLowerCase();
3
Seleccionar todos: const filas = document.querySelectorAll('tr');
4
Filtrar visibilidad: fila.style.display = coincide ? '' : 'none';

🎯 Métodos de Array Útiles

forEach()

array.forEach((item, index) => { console.log(item, index); });

✓ Itera sobre cada elemento

✗ No retorna valor

filter()

const filtrados = array.filter(item => item.nombre.includes('Juan') );

✓ Retorna nuevo array

✓ No modifica el original

map()

const nombres = array.map(item => item.nombre );

✓ Transforma cada elemento

✓ Retorna nuevo array

find()

const encontrado = array.find(item => item.id === 5 );

✓ Retorna primer coincidencia

✓ Retorna undefined si no hay

🚀 Tips y Buenas Prácticas

🎨 CSS vs JavaScript: Usa clases CSS para estilos, JavaScript solo para lógica
⚡ Rendimiento: Evita manipular el DOM en bucles intensivos. Crea elementos y luego inserta todo
🔒 Seguridad: Usa textContent o createTextNode() en lugar de innerHTML con datos del usuario
🐛 Debugging: Usa console.log() para verificar que tus elementos se crean correctamente
📱 Responsive: Prueba tu código en diferentes tamaños de pantalla
♿ Accesibilidad: Usa elementos semánticos (button, a, form) siempre que sea posible

🔧 Ejemplo Completo: Agregar Contacto

// 1. Obtener referencias
const btnAgregar = document.getElementById('btnAgregar');
const lista = document.getElementById('lista');

// 2. Escuchar evento
btnAgregar.addEventListener('click', () => {
  // 3. Crear elementos
  const item = document.createElement('li');
  const texto = document.createTextNode('Nuevo contacto');
  const btnBorrar = document.createElement('button');
  
  // 4. Configurar evento del botón
  btnBorrar.addEventListener('click', () => {
    item.remove();
  });
  
  // 5. Ensamblar
  item.appendChild(texto);
  item.appendChild(btnBorrar);
  lista.appendChild(item);
});
