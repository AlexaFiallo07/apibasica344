Tablas Dinámicas

⭐⭐ Intermedio

🎯 Objetivos Principales

  • Usar forEach() para iterar arrays
  • Crear estructuras complejas (table, tr, td)
  • Mantener referencias para eventos futuros
  • Entender closures con referencia de "fila"
💡 Concepto Clave: forEach()

forEach() ejecuta una función para cada elemento del array. Es ideal para crear múltiples elementos dinámicamente.

const contactos = [ { nombre: "Juan Pérez", tel: "555-0101" }, { nombre: "María García", tel: "555-0102" } ]; contactos.forEach((contacto) => { const fila = document.createElement('tr'); const celdaNombre = document.createElement('td'); celdaNombre.appendChild( document.createTextNode(contacto.nombre) ); fila.appendChild(celdaNombre); tabla.appendChild(fila); });
⚠️ Punto Clave: Mantener referencias

Guardar "fila" como variable permite que cuando hagas clic en el botón, se elimine solo esa fila. Esto es un closure.

Búsqueda y Filtrado en Tiempo Real

⭐⭐ Intermedio

🎯 Objetivos Principales

  • Usar evento 'input' vs 'change'
  • Implementar búsqueda de texto (includes)
  • Manipular visibilidad (display: none)

'input' (Recomendado para búsqueda)
✓ Se dispara cada letra escrita
✓ Búsqueda en tiempo real
✗ Más eventos disparados
'change' (Después de terminar)
✓ Solo cuando pierdes el foco
✓ Menos eventos
✗ No es tiempo real

buscador.addEventListener('input', function() { const busqueda = buscador.value.toLowerCase(); const filas = document.querySelectorAll('tr'); filas.forEach(fila => { const nombre = fila.querySelector('td').innerText.toLowerCase(); if (nombre.includes(busqueda)) { fila.style.display = ""; // Muestra } else { fila.style.display = "none"; // Oculta } }); });
🎯 Ventaja de display: "none"

No elimina los elementos, solo los oculta. Los datos siguen en el DOM, así que puedes mostrarlos nuevamente.
